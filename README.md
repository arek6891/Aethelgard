# Aethelgard

A web-based RPG adventure game built with vanilla JavaScript, HTML, and CSS.

## 🎮 Play Online
[**Play Aethelgard Here**](https://arek6891.github.io/Aethelgard/)

## 📖 About
Aethelgard is a role-playing game featuring:
- **Biomes**: Explore different environments like the Uytek and Infernal lands.
- **Combat**: Fight enemies with a cooldown-based combat system.
- **Inventory Service**: Manage your loot with a Diablo-style inventory.
- **Save System**: Your progress is saved automatically.

## 🛠️ Technology Stack
- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: LocalStorage for save data
- **Audio**: Web Audio API

## 🚀 Deployment
This project is hosted on GitHub Pages. To deploy your own version:
1. Fork or clone this repository.
2. Go to **Settings** > **Pages**.
3. Select `main` branch as source and save.
